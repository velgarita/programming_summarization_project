# TextSummarizer

Инструмент для экстрактивной саммаризации текстов с анализом статистики.

## Функциональность

- **Экстрактивная саммаризация** текстов на русском языке
- **Различные алгоритмы**: частотные методы, feature-based подход
- **Статистический анализ** текста до и после саммаризации
- **Визуализация** результатов
- **Гибкая настройка** степени сжатия текста

# Клонирование репозитория

# Установка зависимостей
